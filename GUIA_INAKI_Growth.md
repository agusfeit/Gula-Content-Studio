## Contexto del proyecto: cómo está configurado el estudio

El proyecto en Claude está montado para que el asistente actúe como **directora creativa, estratega de contenido, guionista y partner para campañas con marcas**. El contenido de Gula mezcla humor, lifestyle, storytelling, tendencias, vínculo humano-perro y UGC.

Reglas no negociables del estudio (útiles para entender por qué se decide lo que se decide):

1. **Identidad antes que oportunidad.** Si una campaña empuja a Gula a ser otra cosa, se negocia o se rechaza.
2. **TikTok-native siempre.** Lo que no se filma fácil con un celular y un humano, se replantea.
3. **Tres opciones a cada marca.** Una segura, una sweet spot, una con upside viral.
4. **Aprender de cada pieza.** Lo que funciona y lo que no se registra; la carpeta se actualiza.
5. **El primer segundo es sagrado.** Sin hook fuerte, no hay video.

Tono: natural, moderno, divertido, auténtico, emocional cuando hace falta; nunca acartonado ni demasiado publicitario.

## Cómo mantenerte actualizado

Este repo cambia seguido. Para tener siempre la última versión:

```bash
# Primera vez (clonar):
git clone <URL-del-repo>
cd "Gula Content Studio"

# Cada vez que querés lo último:
git pull
```

Si conectás este repo a tu propio proyecto de Claude (o lo clonás en tu carpeta de trabajo), tu especialista de growth puede leer cualquier archivo de acá como contexto. Tras un `git pull`, ve los cambios nuevos.

> Las métricas de Instagram se cargan a mano (no hay conector con Meta todavía). Mirá `06_Metricas_Instagram/README.md` para el detalle de cómo y cada cuánto se actualizan.
