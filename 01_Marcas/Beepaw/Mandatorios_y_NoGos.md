# Mandatorios y No-Gos — Beepaw

> Reglas creativas validadas + restricciones de la marca. Esta lista pisa cualquier decisión de guion.

## Mandatorios estructurales (siempre)
- [ ] El contenido se siente **genuino**, nunca publicidad tradicional.
- [ ] **Problema primero**, producto después.
- [ ] Tono **conversacional, primera persona**, como una dueña real.
- [ ] Humor cotidiano + complicidad con el espectador.
- [ ] **Mostrar más de lo que se explica.**
- [ ] Producto integrado **naturalmente** a una escena de uso real.
- [ ] Cierre con frase **humana / divertida**, no comercial.
- [ ] Tags / handles: _completar_
- [ ] Hashtags fijos: _completar_

## Decálogo para futuros guiones

### SIEMPRE
1. Empezar con problema o identificación.
2. Hablar como experiencia personal.
3. Mantener frases cortas.
4. Priorizar humor y realidad.
5. Mostrar más de lo que se explica.
6. Integrar el producto naturalmente.
7. Cerrar con frase humana / divertida.

### NUNCA
- Sonar vendedor.
- Empezar con specs.
- Exagerar claims.
- Usar tono corporativo.

## Mandatorios por producto

| Producto | Mandatorios específicos |
|----------|--------------------------|
| Bubbl-e | Mostrar espuma visualmente satisfactoria + transformación antes/después + suciedad real al inicio |
| Funda para auto | Mostrar caos previo del auto + beneficios funcionales (engancha adelante y atrás, no se mueve, cubre todo, abertura cinturón, protección puertas, comodidad del perro) integrados al uso, no listados |

## No-Gos (cosas a evitar)
- **Lenguaje corporativo.**
- **Claims exagerados.**
- **Frases marketineras.**
- **Demasiadas especificaciones técnicas.**
- Sonar a **review sponsoreada**.
- **CTA de venta directa** ("comprala", "link en bio").
- **Exceso de beneficios** apilados.
- Tono demasiado "perfecto".
- Hablar del producto **antes** del problema.
- Guiones largos y explicativos.
- Contenido demasiado prolijo / artificial.
- Humor agresivo.

## Patrones que ya fracasaron
- *"Invertí en una buena funda"* → sonaba publicitario, bajaba autenticidad, perdía humor.
- Explicar muchos beneficios técnicos → rompe ritmo, baja retención, se siente anuncio.
- CTA directos tipo *"Comprala"* / *"Link en bio"* → rompe naturalidad, baja engagement orgánico.

## Estilo visual obligatorio
- Real, espontáneo, casero, natural.
- **No** ultra producido.
- Handheld funciona bien.
- Planos cercanos, expresiones de perros.
- Lo visual más importante: **transformación + caos real + suciedad visible + comodidad del perro.**
