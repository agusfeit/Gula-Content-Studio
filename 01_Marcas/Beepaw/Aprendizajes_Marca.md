# Aprendizajes — Beepaw

> Insights estratégicos centrales + saber operativo de cómo se trabaja con ellos.

## Insights estratégicos centrales

### 1. Insight principal — el público YA vive el problema
No hace falta explicarlo técnicamente. Hay que **mostrarlo**: caos, suciedad, frustración, incomodidad.

### 2. Insight visual — si el problema se VE, no hace falta describirlo
La cámara hace el trabajo del copy. Cuanto más visible es el problema en el primer frame, menos hay que explicar después.

### 3. Insight emocional — la gente conecta con honestidad e imperfección
Conecta con: honestidad, imperfección, vida real.
**NO** con: contenido ultra producido.

### 4. Insight sobre productos — beneficios funcionales > especs técnicas
Los beneficios fuertes son: practicidad, comodidad, menos caos, mejor experiencia diaria.
**NO** especificaciones técnicas.

### 5. Diferencial de la campaña
> *"La vida real con perros contada de forma divertida y honesta."*

El diferencial NO es el producto solo. Es el ángulo narrativo.

---

## Qué tipo de pieza les funciona mejor
- Hooks relatables con problema visible.
- Estructura Problema → Frustración → Producto → Solución → Resultado → Remate humano.
- Antes/después visual fuerte (auto sucio → limpio, perro mugroso → limpio).
- Humor + transformación juntos.
- Cierre humano sin venta directa.

## Qué tipo de pieza les funciona peor
- Hablar del producto antes del problema.
- Listado de beneficios técnicos.
- CTA de venta directa.
- Guiones largos / explicativos.
- Tono "perfecto" o premium aspiracional.
- Lenguaje corporativo.
