# Cómo usar esta plantilla

Para cada marca nueva (recurrente o one-shot que valga la pena trackear):

1. **Duplicar** esta carpeta `_Plantilla_Marca` y renombrarla con el nombre de la marca (ej. `Pedigree`, `Royal_Canin`, `Petco`).
2. Completar los 5 archivos base:
   - `Brief_Marca.md` — quiénes son, cómo trabajan, por qué encajan con Gula.
   - `Mandatorios_y_NoGos.md` — qué tiene que cumplir siempre, qué no se puede tocar.
   - `Historial_Campanas.md` — todo lo que hicimos juntos.
   - `Guiones_y_Conceptos.md` — ideas vivas + archivadas.
   - `Aprendizajes_Marca.md` — saber operativo de cómo trabajar con ellos.
3. Abrir un **chat dedicado a la marca** en Cowork con el mismo nombre.
4. En el primer mensaje del chat, pedirme que lea esta carpeta para tener el contexto cargado.

## Diferencia entre carpeta y chat
- **Carpeta** = memoria persistente, base de verdad de la marca.
- **Chat** = espacio de trabajo activo donde producimos ideas, guiones y feedback.

La carpeta alimenta al chat. El chat actualiza la carpeta cuando hay algo nuevo que vale la pena guardar.

## Para one-shots chicos
Si la marca es un one-shot que no esperamos repetir, en lugar de carpeta propia, usar `05_One_Shots/` con un solo archivo por campaña.
