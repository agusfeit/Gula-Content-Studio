# Gula Content Studio

Estudio creativo centralizado para Gula — Border Collie merle, creadora de contenido en TikTok / Reels / Instagram.

Este folder es la **memoria persistente** del proyecto. Los chats de Cowork son los **espacios de trabajo** que se apoyan en esta memoria.

---

## Estructura

```
Gula Content Studio/
├── 00_Identidad_Gula/         → Quién es Gula, cómo se ve, cómo habla, sus métricas
├── 01_Marcas/                 → Una carpeta por marca recurrente (a partir de _Plantilla_Marca)
├── 02_Biblioteca_Creativa/    → Hooks, formatos, tendencias, banco de ideas, aprendizajes
├── 03_Plantillas/             → Plantillas reutilizables (brief, guion, caption, propuesta)
├── 04_Campañas_Activas/       → Campañas en producción ahora (carpeta por campaña)
├── 05_One_Shots/              → Colaboraciones únicas que no abren chat propio de marca
└── README.md                  → Este archivo
```

## Mapa de chats sugerido en Cowork

La lógica: **un chat por marca recurrente** + **chats transversales** que cruzan todas las marcas.

### Chats transversales (siempre abiertos)
1. **Gula — Base General** _(este chat)_
   Decisiones de identidad, mapa de chats, estructura, dudas estratégicas.
2. **Gula — Orgánico**
   Ideas y guiones para contenido sin marca. Banco de ideas activo.
3. **Gula — Tendencias y Cultura**
   Radar de trends, audios, formatos. Curaduría semanal.
4. **Gula — One-Shots**
   Campañas únicas con marcas que no abren chat propio.

### Chats por marca recurrente
Un chat por cada marca recurrente activa:
- **Gula — Tonipets**
- **Gula — Crystal Pet Lovers**
- **Gula — Lavakan**
- **Gula — Beepaw**
- **Gula — Happet**

Cada chat de marca abre con la instrucción de leer la carpeta `01_Marcas/[Marca]/` para cargar contexto.

### Chats temporales (se cierran al terminar)
- **Gula — Campaña [Nombre]** — mientras dura una campaña intensa que merece foco aislado.

## Cómo abrir trabajo nuevo

**Brief nuevo de una marca recurrente:**
1. Ir al chat de esa marca.
2. Pedirme que lea `01_Marcas/[Marca]/` + el brief recibido.
3. Interpretar el brief con `03_Plantillas/Plantilla_Brief.md`.
4. Proponer ideas con `03_Plantillas/Plantilla_Propuesta_Marca.md`.

**Brief de una marca nueva:**
1. Duplicar `01_Marcas/_Plantilla_Marca/` → renombrar con el nombre de la marca.
2. Abrir chat dedicado.
3. Cargar brief y empezar.

**Idea para orgánico:**
1. Ir al chat "Gula — Orgánico".
2. Guardar ideas validadas en `02_Biblioteca_Creativa/Banco_de_Ideas.md`.

**Trend que vimos en otro lado:**
1. Ir al chat "Gula — Tendencias y Cultura".
2. Registrar en `02_Biblioteca_Creativa/Tendencias_Activas.md`.

## Reglas de oro del estudio

1. **Identidad antes que oportunidad.** Si una campaña empuja a Gula a ser otra cosa, se negocia o se rechaza.
2. **TikTok-native siempre.** Lo que no se filma fácil con un celular y un humano, replantear.
3. **Tres opciones a la marca.** Una segura, una sweet spot, una con upside viral.
4. **Aprender de cada pieza.** Registrar lo que funciona y lo que no. La carpeta se actualiza.
5. **El primer segundo es sagrado.** Sin hook fuerte, no hay video.
