# Banco de Ideas — Gula

> Pool general de ideas para contenido orgánico. Una idea por entrada, lista para tomar cuando haga falta llenar el calendario.

## Cómo se usa
- Cada idea con un título corto + descripción de 2-3 líneas.
- Etiquetar por **formato** y **categoría**.
- Marcar **estado**: idea cruda / desarrollada / filmada / publicada / archivada.

## Categorías
- Humor cotidiano
- Vínculo humano-perro
- Inteligencia Border Collie
- Tutoriales / "le enseñé a Gula a…"
- Storytime
- Reacciones
- POV / voz interna
- Trends adaptados
- Día en la vida
- Opinión / declaración
